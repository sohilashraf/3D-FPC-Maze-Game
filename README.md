# üß© 3D FPC Maze Game (OpenGL + SFML)

A **3D First-Person Camera (FPC) maze game** built using **OpenGL**, **SFML**, **GLEW**, and **GLM**.  
The player explores a 3D maze from a first-person perspective, navigates around walls with collision detection, and reaches a rotating goal object to win.

This project focuses on:

- First-person camera movement  
- Mouse-look camera control  
- Collision detection  
- Basic lighting and texturing  

---

## üéÆ Gameplay Overview

- Move through a 3D maze using a first-person camera  
- Walls block movement using collision detection  
- A rotating 3D goal marks the maze exit  
- Reaching the goal ends the game
